gishub push <url>

  Pushes the current checkout to the given url.

Parameters:

  <url> (required)

    Push to the given URL, which must give access to a dat repository.

Options:

  --username (-u) / --password (-p)

    Give a username and/or password for ssh or http basic auth connections.

  --live

    Keep pushing streaming changes in real-time as newline-delimited JSON.
