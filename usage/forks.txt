gishub forks

  Returns a newline-delimited stream of forks in the dat. A fork is a version that is incompatible with another version. Forks can be merged with `gishub merge`.

Options:

  --json

    Return results to stdout as a json-parsable string.

    Example:
    {
      "forks":[
        "a2c77bedfb59f2237825614b57109ef3",
        "5e097f5b1bf9320e9a52ef90768d566a"
      ]
    }
